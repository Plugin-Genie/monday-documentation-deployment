Integrate Power Automate and monday.com

Integrate Power Automate with monday.com in less than 5 minutes.

Create a new Power Automate flow using the monday.com connector​

To get started, we will create a new Power Automate flow based on trigger. We'll authenticate the connector with monday.com. Finally, we'll use the output of the trigger in a Power Automate action.

1. In Power Automate, click on "My flows" in the left sidebar​

2. Click the "New flow" button​

3. Select "Automated cloud flow"​

4. Click the blue Skip button on the bottom​

5. Search for "monday"​

6. Select the trigger "When an item is created"​

7. Click the "Sign in" button to authenticate Power Automate with monday.com​

8. A new small window will pop up. Scroll down and click the "Authorize" button. This allows Power Automate to access and create data in monday.com.​

9. Select your workspace and board. When an item is created on this board, it will run the trigger.​

10. Add whichever steps you would like to execute when the trigger is triggered, using the data from your monday.com trigger.​

For example, here we've selected Excel's "Add a row into a table" action to add the item's data into an Excel spreadsheet.

You can include the item data from the "When an item is created" trigger in your actions. In this example, we've selected the Item Name and are also including the item's Status column.

11. Finally, save the flow by clicking the Save button.​

When you create an item in monday.com, it will now execute your Power Automate flow.
