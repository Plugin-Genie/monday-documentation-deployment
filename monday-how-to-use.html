Prerequisites

You will need an active Power Automate account and access to premium plugins. If you don't have a Power Automate account, you can follow these directions to sign up for one: Sign up and sign in for Power Automate.

Getting Started

Add the Power Automate App Usage Board View

1. In a new or existing board, click the plus button to create a new board view.

2. Click the "Apps" option in the menu.

3. Click the "Open in board" button for the Power Automate App Usage board view.

4. The Power Automate App Usage board view has now been added to your board.

This board view initially has information on how to set up the integration. It also includes a video at the bottom showing how to add a monday action/trigger to a Power Automate flow.

Once you've executed a monday action/trigger in Power Automate, navigate back to this board view and refresh the page to view the usage of the integration. This is helpful to make sure you're under your subscription limit. It also allows upgrading your subscription if needed.

Create a Flow in Power Automate

In this section, we will show how to create a new Power Automate flow using the "When an item is created" monday trigger, then save the item name to an Excel spreadsheet.

If instead you would like to create an action instead of a trigger, please follow our documentation for how to do this.

1. Navigate to Power Automate.

You can also access this link from within monday via the Power Automate App Usage board view.

2. In Power Automate, click on "My flows" in the left sidebar

3. Click the "New flow" button

4. Select "Automated cloud flow"

5. Add a name for your flow. Then search for "monday" in the search box.

6. Select the trigger "When an item is created", then click the "Create" button.

7. Click the "Sign in" button to authenticate Power Automate with monday.com

Please note, for users with multiple monday accounts - a separate monday Power Automate connector is needed for each monday account.

8. A new small window will pop up. Scroll down and click the "Authorize" button. This allows Power Automate to access and create data in monday.com.

9. Select your workspace and board. When an item is created on this board, it will run the trigger.

10. Add whichever steps you would like to execute when the trigger is triggered, using the data from your monday.com trigger.

For example, here we've selected Excel's "Add a row into a table" action to add the item's data into an Excel spreadsheet.

Click inside the Name field, which will then automatically open the "Dynamic content" window. In this window, you can select the fields that are returned by the trigger. In this case, we're selecting the Item Name field.

Note: Other monday triggers, such as the "When a column changes" trigger, contain a lot more dynamic content (such as the columns for the item).

11. Finally, save the flow by clicking the Save button.

When you create an item in monday.com, it will now execute your Power Automate flow, which will save the name of the created item to Excel.
